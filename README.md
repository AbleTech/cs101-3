##  Setup

Install required rubygems:
```sh
$ bundle install
```

Install required JavaScript libraries:
```sh
$ npm install -g bower
$ bower install
```

Start Middleman server:
```sh
$ bundle exec middleman server
```

You can also symlink this project to your `.pow` directory.
